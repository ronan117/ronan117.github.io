Understanding and using Greater Than, and Less Than

Page 1   Page 2   Page 3

What are they?

  • They're called inequalities
    1. The Greater Than symbol is >
    2. The Less Than symbol is <
  • They're called inequalities
    1. The definition is
      • The relation between two expressions that are not equal
You can use the two in equations to signify if a number is Greater than or Less Than another number.

How would you use it in an equation?

  • 1 > 0, 1 is Greater than 0 because it is a bigger number
  • 0 < 1, 0 is Less Than 1 because it is a smaller number.
